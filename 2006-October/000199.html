About your text rendering problem, try not using GlyphKeeper and use the 
internal renderer. When compiling the code pass -DUSE_NEW_TTF and omit 
linking to glyphkeeper. You do need to make sure you have freetype2 installed 
properly and just link against that. If you like you can can make a patch of 
your changes on the source tree, attach it here and I'll test them out for 
you on gcc.
